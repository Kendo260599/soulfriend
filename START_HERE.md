### Common Issues:

**"GitHub push failed"**
```
Solution:
1. Create repo at github.com/new first
2. Name it: soulfriend
3. Run script again
```

**"Vercel login failed"**
```
Solution:
1. Browser will open
2. Click "Continue with GitHub"
3. Authorize Vercel
```

**"Render is manual?"**
```
Yes! Render doesn't have full CLI yet.
Takes 5 minutes in dashboard:
1. Go to: dashboard.render.com
2. New Web Service
3. Connect GitHub repo
4. Click Deploy
Done!
```
