🔄 Force Clear Vercel Cache & Test AI

Mục đích: Clear toàn bộ cache frontend và test chatbot AI

📋 STEP 1: Clear Browser Cache

Làm TẤT CẢ các bước này:

  1. Nhấn Ctrl + Shift + Delete
  2. Chọn "All time" hoặc "Mọi lúc"
  3. Check:
    • ✅ Cookies and site data
    • ✅ Cached images and files
    • ✅ Hosted app data (nếu có)
  4. Click "Clear data"

📋 STEP 2: Force Reload Vercel

Mở Vercel app và:

  1. Nhấn Ctrl + F5 (hard reload)
  2. HOẶC nhấn Ctrl + Shift + R
  3. HOẶC mở Incognito: Ctrl + Shift + N

📋 STEP 3: Test Backend AI Trực Tiếp

📋 STEP 4: Test từ Vercel Frontend

Sau khi clear cache & reload:

  1. Mở chatbot trong Vercel app
  2. Gửi message: "Xin chào, bạn là ai?"
  3. Check Network tab (F12) → filter "message"
  4. Xem Response có aiGenerated: true không

🔍 Troubleshooting

Nếu vẫn "offline":

  1. Check Vercel env vars: REACT_APP_API_URL
  2. Redeploy Vercel (không dùng cache)
  3. Check frontend code xử lý response
